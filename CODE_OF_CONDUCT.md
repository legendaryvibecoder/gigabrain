# Code of Conduct

## Our Standard

We want Gigabrain to be a calm, collaborative project where contributors can ask questions, share ideas, and improve the system together.

Please:

- be respectful and specific
- assume good intent, but be honest about problems
- focus critique on behavior, code, or design, not on people
- make room for learners and first-time contributors

## Unacceptable Behavior

This project does not allow:

- harassment or personal attacks
- hateful or discriminatory language
- intimidation, dogpiling, or sustained hostility
- doxxing or sharing private information without permission
- deliberately disruptive behavior in issues, discussions, or pull requests

## Enforcement

Project maintainers may moderate discussions, close threads, or remove contributions that break these standards.

For serious concerns, please open a private security advisory if it is security-related, or contact the maintainers through GitHub if it is conduct-related.
