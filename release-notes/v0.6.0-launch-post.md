# Gigabrain v0.6.0

Gigabrain `v0.6.0` is live.

This is the first release where recall quality is something we can actually measure instead of tuning by feel.

What is new:

- real recall evaluation with a new in-process eval harness, nightly quality history, and latency metrics
- smarter recall with BM25 ranking, adaptive budgets, confidence labels, and optional semantic reranking
- stronger OpenClaw, Codex, Claude Code, and Claude Desktop stability, including fixed OpenClaw hook behavior, empty-store bootstrap, dynamic helper resolution, and a hardened Claude Desktop launcher bundle
- more world-model visibility through `/gb/evolution`, `/gb/relationships`, and new MCP tools for `gigabrain_entity`, `gigabrain_contradictions`, and `gigabrain_relationships`

This release also resolves the public OpenClaw correctness issues [`#40`](https://github.com/legendaryvibecoder/gigabrain/issues/40), [`#41`](https://github.com/legendaryvibecoder/gigabrain/issues/41), and [`#42`](https://github.com/legendaryvibecoder/gigabrain/issues/42).

Install:

```bash
npm install @legendaryvibecoder/gigabrain
```
