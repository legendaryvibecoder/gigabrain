# Memory Passport Pilot Offer

## Headline

AI agents remember more than teams can see. Gigabrain gives you a local Memory Passport before those memories become operational risk.

## Offer

For teams using two or more AI coding agents, Gigabrain will run a first-pass memory audit and deliver:

- source inventory across visible Codex, Claude Code, Cursor/Windsurf, OpenClaw, and manual cloud exports
- duplicate memory groups
- stale or expired facts
- contradiction review items
- provenance gaps
- secret-risk rows with redacted previews
- safe AGENTS.md and CLAUDE.md handoff briefs with secret-risk rows omitted
- a Passport readiness score and blocker list

## Buyer Pain

- "Claude remembers one version, Codex remembers another."
- "Nobody knows whether our agents picked up secrets from a local note."
- "We need to hand off agent context without dumping everything into another model."
- "Security asked what these agents remember and we do not have an answer."

## Qualification

Good fit:

- team actively using Codex, Claude Code, Cursor/Windsurf, ChatGPT, or OpenClaw
- at least one repo where AI agents have durable instructions or memories
- concern about secrets, stale facts, provenance, or handoff hygiene

Bad fit:

- single assistant only
- no durable memory/instructions
- wants cloud scraping of closed products
